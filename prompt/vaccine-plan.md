# 📅 ตารางการฉีดวัคซีนสำหรับสุนัขและแมว (Vaccination Schedule)

## 🐱 1. โปรแกรมวัคซีนแมว (Cat Vaccination)

### **วัคซีนหลัก (Core Vaccines)**
จำเป็นต้องฉีดสำหรับแมวทุกตัว

#### 🅰️ วัคซีนรวมหัด-หวัดแมว (FVRCP)
ป้องกัน: ไข้หัดแมว, หวัดแมว (Herpesvirus, Calicivirus)

| เข็มที่ | อายุที่แนะนำ | ระยะห่างจากเข็มก่อนหน้า |
| :--- | :--- | :--- |
| **1** | 8 สัปดาห์ (2 เดือน) | - |
| **2** | 12 สัปดาห์ (3 เดือน) | 4 สัปดาห์ (1 เดือน) |
| **3** | 16 สัปดาห์ (4 เดือน) | 4 สัปดาห์ (1 เดือน) |
| **กระตุ้น** | ทุก 1 ปี | ห่างจากปีที่แล้ว 1 ปี |

#### 🅱️ วัคซีนพิษสุนัขบ้า (Rabies)
ป้องกัน: โรคพิษสุนัขบ้า (กฎหมายบังคับ)

| เข็มที่ | อายุที่แนะนำ | ระยะห่างจากเข็มก่อนหน้า |
| :--- | :--- | :--- |
| **1** | 12 สัปดาห์ (3 เดือน) | - |
| **2** | 16 สัปดาห์ (4 เดือน) | 4 สัปดาห์ (1 เดือน) |
| **กระตุ้น** | ทุก 1 ปี | ห่างจากปีที่แล้ว 1 ปี |

### **วัคซีนทางเลือก (Optional Vaccines)**
ฉีดตามความเสี่ยง (เช่น เลี้ยงปล่อย, เลี้ยงรวมหลายตัว)

* **วัคซีนลูคีเมีย (FeLV):** เริ่มฉีดที่อายุ 8 สัปดาห์ขึ้นไป ฉีด 2 เข็ม ห่างกัน 3-4 สัปดาห์ (ควรตรวจเลือดก่อนฉีด)
* **วัคซีนเอดส์แมว (FIV):** ปัจจุบันไม่ค่อยนิยมฉีด
* **วัคซีนเยื่อบุช่องท้องอักเสบ (FIP):** แบบหยอดจมูก เริ่มที่อายุ 16 สัปดาห์

## 🐶 2. โปรแกรมวัคซีนสุนัข (Dog Vaccination)

### **วัคซีนหลัก (Core Vaccines)**
จำเป็นต้องฉีดสำหรับสุนัขทุกตัว

#### 🅰️ วัคซีนรวม 5 โรค (DHPPL)
ป้องกัน: ไข้หัด, ลำไส้อักเสบ, ตับอักเสบ, หวัด, ฉี่หนู

| เข็มที่ | อายุที่แนะนำ | ระยะห่างจากเข็มก่อนหน้า |
| :--- | :--- | :--- |
| **1** | 6-8 สัปดาห์ (1.5 - 2 เดือน) | - |
| **2** | 10-12 สัปดาห์ (2.5 - 3 เดือน) | 3-4 สัปดาห์ |
| **3** | 14-16 สัปดาห์ (3.5 - 4 เดือน) | 3-4 สัปดาห์ |
| **กระตุ้น** | ทุก 1 ปี | ห่างจากปีที่แล้ว 1 ปี |

#### 🅱️ วัคซีนพิษสุนัขบ้า (Rabies)
ป้องกัน: โรคพิษสุนัขบ้า (กฎหมายบังคับ)

| เข็มที่ | อายุที่แนะนำ | ระยะห่างจากเข็มก่อนหน้า |
| :--- | :--- | :--- |
| **1** | 12 สัปดาห์ (3 เดือน) | - |
| **2** | 16 สัปดาห์ (4 เดือน) | 4 สัปดาห์ (1 เดือน) |
| **กระตุ้น** | ทุก 1 ปี | ห่างจากปีที่แล้ว 1 ปี |

### **วัคซีนทางเลือก (Optional Vaccines)**

* **วัคซีนหลอดลมอักเสบติดต่อ (Kennel Cough):** เริ่มฉีดได้ตั้งแต่อายุ 6-8 สัปดาห์ (แนะนำสำหรับสุนัขที่เข้าสังคมบ่อย หรือต้องไปโรงแรมสุนัข)
* **วัคซีนไข้หวัดใหญ่สุนัข (Canine Influenza):** ฉีด 2 เข็ม ห่างกัน 2-4 สัปดาห์

## 💡 หมายเหตุสำคัญ (Note)
1.  **ถ่ายพยาธิ:** ควรเริ่มถ่ายพยาธิพร้อมกับการฉีดวัคซีนเข็มแรก และทำซ้ำทุกเดือนจนอายุ 6 เดือน
2.  **การสังเกตอาการ:** หลังฉีดวัคซีน สัตว์เลี้ยงอาจมีอาการซึมหรือมีไข้ต่ำๆ ได้ 1-2 วัน หากมีอาการหน้าบวม อาเจียน หรือแพ้รุนแรง ควรรีบพากลับไปพบสัตวแพทย์ทันที
3.  **สมุดวัคซีน:** ห้ามทำหาย ต้องนำไปทุกครั้งที่ฉีดวัคซีนเพื่อบันทึกประวัติและยี่ห้อวัคซีน
